fastapi——快速入门

Posted on 2019-09-26 Edited on 2021-06-10 In python

fastapi是高性能的web框架。他的主要特点是：
- 快速编码
- 减少人为bug
- 直观
- 简易
- 具有交互式文档
- 基于API的开放标准（并与之完全兼容）：OpenAPI（以前称为Swagger）和JSON Schema。

技术背景：python3.6+、Starlette、Pydantic
